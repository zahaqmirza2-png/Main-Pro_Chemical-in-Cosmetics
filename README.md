## Project Title
**Cosmetic Products Risk & Discontinuation Analysis using Pandas and Matplotlib**

---

## 1. Project Overview

This project focuses on analyzing cosmetic product data to understand **chemical risk levels**, **product discontinuation patterns**, **company-wise distributions**, and **primary product categories**.

The analysis uses Python data analysis libraries to identify trends related to **high-, medium-, and low-risk chemicals** and how they influence **product lifecycle decisions**.

The goal of this project is to extract **meaningful insights** that can support **product safety evaluation, regulatory compliance, and strategic decision-making** in the cosmetics industry.

## 4. Steps Followed

1. Imported the dataset using **Pandas**  
2. Cleaned the data by removing missing and duplicate values  
3. Standardized categorical and numerical fields  
4. Performed **Exploratory Data Analysis (EDA)**  
5. Grouped data by:
   - Company  
   - Product category  
   - Chemical risk level  
   - Discontinued status  
6. Created visualizations using **Matplotlib**  
7. Interpreted results to extract meaningful insights  

---

## 5. Key Insights

- Makeup products (non-permanent) show the highest number of discontinued products  
- Companies with larger product portfolios experience higher discontinuation counts  
- Low-risk products dominate discontinuation numbers, indicating business-driven decisions  
- High-risk discontinued products are fewer but concentrated among specific companies  
- Discontinuation trends are influenced more by category dominance and scale than risk alone
